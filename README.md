# Intern Technical Assessment

This repository contains a collection of technical assessment projects completed as part of internship evaluations and skill-building challenges. Each project is separated into its own folder with clear organization and purpose.

The projects showcase practical experience and demonstrate competencies in **Full Stack Development**, **Backend API design**, **Frontend UI**, **Database integration**, and **deployment workflows**.

---

## Projects Included

Below is a quick overview of the projects inside this repository:

### **Creative_Showcase**
A frontend application displaying creative content and UI elements to demonstrate design skills.

### **Smart ToDo API**
A backend RESTful API for managing to-do items, featuring CRUD operations and persistent storage.

### **Spotify Lyric Search**
An application integrating with music APIs to let users find lyrics for songs by searching artist/title.

### **Weekend Gateway Ranker**
A utility app that helps rank weekend getaway destinations based on certain criteria (e.g., distance, popularity, cost).
